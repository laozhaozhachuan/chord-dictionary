# 和弦词典

一个简单的和弦查询网页工具。

## 功能

- **和弦查音名**：输入和弦名称（如 Cmaj7、Dm、G7），显示组成音
- **音名查和弦**：输入一组音名（如 C E G），显示可能的和弦

## 支持的和弦类型

- 三和弦：大三、小三、增三、减三
- 七和弦：大七、小七、属七、减七、半减七
- 挂留和弦：sus2、sus4
- 延伸和弦：九和弦、十一和弦、十三和弦
