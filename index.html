和弦词典

和弦标记说明

  • 大三和弦: C, D, E, F, G, A, B
  • 小三和弦: Cm, Dm, Em, Fm, Gm, Am, Bm
  • 增三和弦: Caug, Daug...
  • 减三和弦: Cdim, Ddim...
  • 七和弦: C7, Cmaj7, Cm7, Cdim7, Cm7b5
  • 挂留和弦: Csus2, Csus4
  • 六和弦: C6, Cm6
  • 九/十一/十三和弦: C9, C11, C13
  • 转位: C/E, C/G (斜杠后为低音)

输入说明

  • 支持英文: C#, Db, C, D, E
  • 支持大小写: C 和 c 都可以
  • 空格或逗号分隔多个音
